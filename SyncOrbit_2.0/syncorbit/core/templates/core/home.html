📐 Architecture

How SyncOrbit Works

A 5-stage autonomous pipeline from raw TLE to zero-loss handover

01

TLE Ingestion

Fetches live Keplerian elements from CelesTrak every few seconds via Skyfield

02

Visibility Scan

Computes elevation + azimuth for all LEO sats relative to ground station in New Delhi

03

LogReg Selector

Logistic Regression scores every candidate on RSSI + elevation → picks the best target

04

LSTM Gating

10-frame sliding window fed to LSTM → confidence score. Fires when ≥ 40%

05

Zero-Loss Switch

Buffer absorbs packets. New link established. Buffer replayed. 0% packet loss

✨ Capabilities

What Makes SyncOrbit Different

Purpose-built for LEO constellation handover with a production-grade ML backbone

🤖

Dual-Model ML Pipeline

Logistic Regression for candidate ranking + LSTM for timing prediction. Two models, one seamless decision.

📡

Real TLE Data

Live Keplerian element ingestion from CelesTrak. Actual satellite physics, not simulated approximations.

📦

Zero Packet Loss

Ground Station B buffer absorbs all in-flight packets during the handover window and replays after switch.

⚡

Predictive — Not Reactive

LSTM fires before signal degrades. Legacy systems wait until RSSI drops below threshold — too late.
